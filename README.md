
# 16S rRNA Bacterial Identification using BLAST

## Project Overview
This project focuses on identifying bacterial species using 16S rRNA gene sequences and BLAST tool from NCBI.

## Methodology
- Collected 16S rRNA sequences
- Used NCBI BLAST (blastn)
- Compared sequences with database
- Analyzed % identity, E-value, and query coverage

## Tools Used
- NCBI BLAST
- Bioinformatics databases (GenBank)

## Results
- Escherichia coli identified with 99.5% identity
- Pseudomonas aeruginosa identified with 98.90
  

## Conclusion
16S rRNA + BLAST is a reliable and cost-effective method for bacterial identification.
